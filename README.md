# Рейтинг ординатуры КФУ 2026

Живая таблица конкурсных списков ординатуры (abit.cfuv.ru) + оценка шансов абитуриента.

**Страница:** https://Keeg4n.github.io/ord/

## Что внутри

| Файл | Назначение |
|------|------------|
| `index.html` | Вьюха: общая таблица (направление · форма · мест · поступающих · макс. балл), поиск, фильтр, блок «Шансы абитуриента» |
| `data.json`  | Собранные данные |
| `collect.php` | Скрапер: обходит все направления ординатуры и все источники → `data.json` |
| `fr_parse.php` | Парсер отчётов FastReport по координатам (зависимость collect.php) |
| `cfuv_rating.php` | Выгрузка одного отчёта по `specialtyId/source/form` |
| `show.php` | Тот же анализ шансов в консоли |

## Обновить данные

Нужен PHP с расширением cURL:

```bash
php collect.php        # тянет свежий список направлений и рейтинги, пишет data.json
git add data.json && git commit -m "refresh data" && git push
```

## Логика «бюджета»

Незанятые целевые места 13 августа переходят в бюджет (общие основания), поэтому
`эффективный бюджет = объявленные бюджетные места + свободные целевые`.

## Источники API

- `GET /api/specialties/enroll/2026/1/8` — список направлений
- `GET /api/freports/list/{spec}/{source}/2` — выдаёт `reportId` (FastReport)
- `POST /_fr/preview.getReport?reportId=...&renderBody=yes` — HTML отчёта

`source`: 1 — общие основания (бюджет), 2 — целевое, 3 — коммерция.
